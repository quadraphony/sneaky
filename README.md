# Sneaky Core

Production-grade Go core for the Sneaky system.

This repository is organized and implemented in controlled phases as defined in `docs/04_PHASE_PLAN.md`.

Current implemented phases:
- Phase 0-1 foundation and core contracts
- Phase 2 config detection and validation
- Phase 3 sing-box adapter foundation
- Phase 4 logging and stats
- Phase 5 CLI tooling
- Phase 6 second adapter introduction with SSH direct SOCKS tunneling
- Phase 7 binding preparation
- Phase 8 hardening

CLI commands:
- `validate`
- `probe`
- `start`
- `stop`
- `status`
- `version`
